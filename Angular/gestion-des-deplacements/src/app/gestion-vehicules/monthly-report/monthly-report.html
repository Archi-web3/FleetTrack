Données environnementales disponibles : Les véhicules avec des données d'émissions CO2 et de consommation permettent de calculer le bilan carbone. Ces informations peuvent être ajoutées dans Gestion Véhicules → Modifier.

• CO2 (kg) = Émissions (g/km) × Distance (km) / 1000
• Carburant (L) = Consommation (L/100km) × Distance (km) / 100
