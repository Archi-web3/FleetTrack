Aide SuperAdmin : Comprendre et Gérer le Workflow

Ce module vous permet de visualiser de bout en bout le processus métier :

  • Lecture : Chaque encart représente une action requise. L'icône person indique qui est responsable de cette étape.
  • Notifications : Les badges bleus (avec l'icône notifications) vous montrent instantanément qui est alerté au cours de cette phase.
  • Paramétrage : Les interrupteurs (Toggles) permettent de désactiver certaines étapes optionnelles (ex: Validation Sécurité) afin d'adapter le flux aux réalités du terrain de chaque Base/Pays. Le paramétrage fin des droits d'action se fera dans la future Matrice des Rôles.
