1. Introduction

FleetTrack est la solution de gestion de flotte optimisée pour ACF, comprenant :

  • Interface Web : Pour gérer les véhicules, planifier les mouvements et configurer la maintenance.
  • E-Logbook Mobile (Chauffeur) : Carnet de bord numérique pour les trajets, pleins de carburant et exécution des services de maintenance.

Rôles Utilisateurs

Rôle Accès Description
Super Admin Interface Web Gestion complète : pays, bases, utilisateurs, véhicules
Admin (Pays) Interface Web Gestion de sa base : utilisateurs, véhicules, mouvements
Superviseur (Base) Interface Web Validation des mouvements et gestion de la flotte de sa base
Superviseur Sécurité Interface Web Validation sécuritaire des mouvements
Technicien (RP/Demandeur) Interface Web Consultation et création de demandes de mouvements (Responsable Programme)
Chauffeur E-Logbook Mobile Enregistrement des trajets, carburant, incidents

2. Accès aux Applications

🌐 Interface Web

URL : https://fleettrack-web.vercel.app

3. Interface Web

3.1 Planning Global

Le planning permet de visualiser tous les mouvements planifiés.

Fonctionnalités :

  • ✅ Visualisation des mouvements à la semaine.
  • ✅ Passez votre souris sur un mouvement pour voir plus de détails (Passagers, Motif, etc.).

Légende des status :

  • 🟡 En attente : Demande soumise.
  • 🟢 Validé : Approuvé par l'administrateur.
  • 🔵 Pris en charge : Chauffeur/Véhicule assigné et confirmé.
  • 🚙 En cours : Trajet démarré.
  • ✅ Terminé : Trajet clôturé.
  • ❌ Annulé : Demande annulée.

3.2 Tableau de Bord

Vue d'ensemble de l'activité du jour et indicateurs clés.

3.3 Nouvelle Demande

Formulaire pour créer un nouveau mouvement.

  • Remplissez les détails : Projet, Date/Heure, Itinéraire, Passagers, Motif.
  • Soumettez pour validation.

3.4 Mes Mouvements

Espace dédié au demandeur (Technicien RP) pour suivre ses propres demandes.

  • Visualiser la liste de vos demandes.
  • Vérifier le statut en temps réel (En attente, Validé, etc.).
  • Consulter l'historique des mouvements terminés.

3.5 Validation

(Réservé aux Admin/Superviseurs)

Interface pour valider ou refuser les demandes de mouvements en attente.

3.6 Consolidation

(Réservé aux Admin/Dispatch)

Outil pour attribuer les véhicules et les chauffeurs aux mouvements validés et organiser le planning.

3.7 Gestion des Véhicules

Gestion des véhicules et de leurs données techniques.

Kilométrage Initial : Lors de l'ajout d'un véhicule, ce champ est crucial. Si modifié sans trajets existants, le kilométrage actuel s'ajuste automatiquement.

3.8 Suivi Maintenance

Module pour assurer le suivi préventif de la flotte.

Onglet Suivi Hebdo

Permet de consulter l'état des checklist hebdomadaires effectuées par les chauffeurs.

Onglet Configuration Intervalles

Définissez les fréquences de maintenance (ex: Vidange tous les 5000 km) pour générer les alertes.

Templates de Service

Configuration des listes de tâches.

Note : Pour l'instant, la référence des instructions techniques provient du manuel MSF et cible principalement les Land Cruiser.

3.9 Administration

Configuration avancée de l'application.

  • Gestion des utilisateurs : Création et gestion des comptes et rôles.
  • Gestion des projets : Codes projets pour l'analytique.
  • Gestion des chauffeurs : Liste spécifique des conducteurs.
  • Gestion des lieux : Points de départ/arrivée fréquents.
  • Paramètres généraux : Configuration globale.
  • Rapport mensuel : Export des données d'activité.

4. E-Logbook Mobile (PWA)

4.1 Utilisation Quotidienne

  • Sélection Véhicule : Choisissez votre véhicule au démarrage.
  • Trajets : Démarrez et arrêtez vos trajets.
  • Carburant : Saisie des pleins.

4.2 Services Programmés (Maintenance)

Exécution des maintenances via l'application.

  1. Allez dans "Maintenance" > "Services Programmés".
  2. Cochez les tâches réalisées.
  3. Utilisez l'icône livre 📖 pour consulter le Manuel Atelier (PDF).
  4. Validez le service une fois terminé.

4.3 Signalement

Utilisez le menu "Incidents" pour déclarer accidents ou pannes.
