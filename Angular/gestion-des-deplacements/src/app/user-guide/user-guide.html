1. Introduction

FleetTrack est la solution de gestion de flotte optimisée pour ACF. Elle repose sur une architecture multi-pays stricte :

  • Cloisonnement des données : Un utilisateur affecté à un pays (ex: Mali) ne peut voir QUE les données de ce pays.
  • Exception SuperAdmin : Seul le rôle "SuperAdmin" possède une vue globale et peut naviguer entre tous les pays.
Important : L'application est divisée en deux interfaces connectées :
  • Interface Web (Admin) : Gestion, Planification, Validation par le bureau.
  • E-Logbook (Chauffeur) : Application Mobile (PWA) pour le terrain.

Rôles Utilisateurs

Rôle Accès Portée (Scope) Responsabilités
SuperAdmin Web (Complet) 🌍 Monde (Tous Pays) Configuration système, création des pays/bases, audit global, gestion des admins pays.
Admin (Pays) Web (Pays) 🏳️ Pays (Mission) Gestion de TOUTES les bases de sa mission. Création des utilisateurs, paramétrage local.
Superviseur (Base) Web (Base) 📍 Base Locale Validation hiérarchique des mouvements, suivi maintenance flotte locale.
Superviseur Sécurité Web (Base/Pays) 🛡️ Transverse Validation sécuritaire des mouvements, envoi des alertes push de sécurité.
Technicien (Demandeur) Web (Restreint) 👤 Personnel Création de demandes de mouvements pour soi ou son équipe.
Chauffeur Mobile PWA 🚙 Véhicule Tenue du carnet de bord, pleins carburant, checklist maintenance.

URL : https://fleettrack-web.vercel.app

📱 E-Logbook Mobile (PWA)

URL : https://fleettrack-mobile.vercel.app

3. Interface Web

3.1 Planning Global

Le cœur operationnel. Visualisez tous les mouvements de la semaine d'un coup d'œil.

Nouveautés :

  • ✅ Drag & Drop : Assignez un véhicule en glissant la demande sur la ressource.
  • ✅ Mode "Inbox" Alertes : Les notifications non lues apparaissent en priorité.

3.2 Vision & Sécurité (Roadmap)

(Nouveau Menu)

Une transparence totale sur l'évolution de l'outil et sa sécurité.

  • Roadmap : Consultez les prochaines fonctionnalités prévues.
  • Changelog : Historique complet des mises à jour.
  • Sécurité : Détails techniques sur la protection des données (RGPD, Chiffrement, Audit).

3.3 Objectifs Environnementaux

Suivez l'empreinte carbone de la flotte.

  • IAP (Indicateur d'Activité Pondéré) : Mesure l'efficacité réelle des déplacements.
  • Plan d'Action : Suivi des objectifs de réduction CO2 (-5% / an).

3.4 Alertes de Sécurité

Permet d'envoyer des notifications urgentes à toute la flotte (via Web Push).

  • Ciblez un véhicule précis ou "TOUTE LA FLOTTE".
  • Le chauffeur reçoit l'alerte même si son téléphone est verrouillé (App fermée).

3.5 Validation & Consolidation

Le workflow de validation à double niveau :

  1. Logistique : Vérifie la disponibilité véhicule.
  2. Sécurité : Valide la faisabilité du trajet (Zones à risques).

Utilisez la Consolidation pour grouper plusieurs demandes dans un même véhicule (Covoiturage).

3.6 Gestion de Flotte & Maintenance

Notifications Maintenance : FleetTrack surveille automatiquement les kilométrages. Un email est envoyé au Superviseur (08h00) si un véhicule approche de son échéance de maintenance.
  • Suivi Hebdo : État des checklists chauffeurs.
  • Configuration : Définissez vos propres intervalles (ex: Vidange LC tous les 5000km).

3.7 Rapports & Analytics

Exportez vos rapports mensuels en un clic (Format standard Logbook).

4. E-Logbook Mobile (PWA)

Mode Hors-Ligne (Offline First) : L'application fonctionne ENTIÈREMENT sans connexion internet. Les données sont stockées dans le téléphone et synchronisées automatiquement dès que le réseau revient (Icône 🔄 verte).

4.1 Notifications Web Push (NOUVEAU)

Pour la sécurité et la réactivité :

  • App Fermée : Vous recevez les alertes de sécurité et les notifications de mission même si l'application est fermée.
  • Activation : Cliquez sur le bouton "Activer Notifications" dans votre Inbox pour autoriser le mobile.

4.2 Utilisation Quotidienne

  • Sélection Véhicule : Choisissez votre véhicule au démarrage.
  • GPS Automatique : Le départ de trajet enregistre votre position GPS précise.
  • Mode Libre : Possibilité de démarrer un trajet imprévu ("Ad-Hoc") sans demande préalable.

4.3 Services & Incidents

  • Maintenance : Cochez les tâches réalisées (avec accès au Manuel Atelier PDF intégré).
  • Incidents : Déclarez accidents ou pannes avec photos à l'appui.
